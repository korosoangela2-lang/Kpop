## K-pop World Website

A dynamic K-pop themed website built using HTML, CSS, and JavaScript (DOM manipulation).


## Description

K-pop World is a fan-based website that includes:

- Information about K-pop
- Popular K-pop groups displayed as dynamic cards
- Image gallery
- Fan registration form

All main elements are created dynamically using JavaScript DOM methods.

##  Features

- Dynamic navigation menu
- Cards generated using JavaScript
- Image gallery section
- Fan registration form
- DOM manipulation
- Responsive layout


## Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- DOM Manipulation


##  Learning Goals

- Practice DOM manipulation
- Create elements dynamically
- Work with arrays and loops
- Build structured frontend projects
